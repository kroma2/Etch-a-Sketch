# Etch-a-Sketch
A simple app that colors each box in the grid with a random color.

## How to use
-   Open `index.html` in your browser
-   Hover your mouse over a box, it will get a random RGB color and get darkened by 10%

## Tech
-   HTML, CSS, JavaScript
